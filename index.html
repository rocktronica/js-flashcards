JavaScript Native Method Flashcards

Array Methods

Array.from()

arr.from(arrayLike[, mapFn, thisArg])

The Array.from() method creates a new Array instance from an array-like or iterable object.

Array.from('foo'); // ["f", "o", "o"]

Array.isArray()

arr.isArray(obj)

The Array.isArray() function determines whether the passed value is an Array.

Array.isArray([1]); // true

Array.of()

arr.of(element0[, element1[, ...[, elementN]]])

The Array.of() method creates a new Array instance with a variable number of arguments, regardless of number or type of the arguments.

Array.of(1, 2, 3); // [1, 2, 3]

Array.prototype.concat()

arr.concat(value1[, value2[, ...[, valueN]]])

The concat() method is used to merge two or more arrays. This method does not change the existing arrays, but instead returns a new array.

['a', 'b', 'c'].concat([1, 2, 3]); // ['a', 'b', 'c', 1, 2, 3]

Array.prototype.copyWithin()

arr.copyWithin(target[, start, end])

The copyWithin() method shallow copies part of an array to another location in the same array and returns it, without modifying its size.

[1, 2, 3, 4, 5].copyWithin(0, 3, 4); // [4, 2, 3, 4, 5]

Array.prototype.entries()

a.entries()

The entries() method returns a new Array Iterator object that contains the key/value pairs for each index in the array.

['a', 'b', 'c'].entries().next().value; // [0, 'a']

Array.prototype.every()

arr.every(callback[, thisArg])

The every() method tests whether all elements in the array pass the test implemented by the provided function.

[1, 30, 39, 29, 10, 13].every(v => v < 40); // true

Array.prototype.fill()

arr.fill(value[, start[, end]])

The fill() method fills all the elements of an array from a start index to an end index with a static value. The end index is not included

[1, 2, 3, 4].fill(0, 2, 4); // [1, 2, 0, 0]

Array.prototype.filter()

arr.filter(callback[, thisArg])

The filter() method creates a new array with all elements that pass the test implemented by the provided function.

[2, 5, 10, 12].filter(v => v >= 10); // [10, 12]

Array.prototype.find()

arr.find(callback[, thisArg])

The find() method returns the value of the first element in the array that satisfies the provided testing function. Otherwise undefined is returned.

[2, 5, 10, 12].find(v => v >= 10); // 10

Array.prototype.findIndex()

arr.findIndex(callback[, thisArg])

The findIndex() method returns the index of the first element in the array that satisfies the provided testing function. Otherwise -1 is returned.

[2, 5, 10, 12].findIndex(v => v >= 10); // 2

Array.prototype.forEach()

arr.forEach((currentValue[, index[, array]]) => {}[, thisArg]);

The forEach() method executes a provided function once for each array element.

[1, 2, 3].forEach(v => { console.log(v * 2); }); // 2, 4, 6

Array.prototype.includes()

arr.includes(searchElement[, fromIndex])

The includes() method determines whether an array includes a certain element, returning true or false as appropriate.

[1, 2, 3].includes(2); // true

Array.prototype.indexOf()

arr.indexOf(searchElement[, fromIndex])

The indexOf() method returns the first index at which a given element can be found in the array, or -1 if it is not present.

['ant', 'bison', 'camel', 'duck', 'bison'].indexOf('bison'); // 1

Array.prototype.join()

arr.join([separator = ','])

The join() method joins all elements of an array (or an array-like object) into a string and returns this string.

['Fire', 'Wind', 'Rain'].join('-'); // Fire-Wind-Rain

Array.prototype.keys()

arr.keys()

The keys() method returns a new Array Iterator object that contains the keys for each index in the array.

for (let key of ['a', 'b', 'c'].keys()) { console.log(key); } // 0, 1, 2

Array.prototype.lastIndexOf()

arr.lastIndexOf(searchElement[, fromIndex])

The lastIndexOf() method returns the last index at which a given element can be found in the array, or -1 if it is not present. The array is searched backwards, starting at fromIndex.

['Dodo', 'Tiger', 'Penguin', 'Dodo'].lastIndexOf('Dodo'); // 3

Array.prototype.map()

arr.map((currentValue[, index[, array]]) => { }[, thisArg])

The map() method creates a new array with the results of calling a provided function on every element in the calling array.

[1, 4, 9, 16].map(x => x * 2); // [2, 8, 18, 32]

Array.prototype.pop()

arr.pop()

The pop() method removes the last element from an array and returns that element. This method changes the length of the array.

['a', 'b', 'c'].pop(); // 'c'

Array.prototype.push()

arr.push(element1[, ...[, elementN]])

The push() method adds one or more elements to the end of an array and returns the new length of the array.

a = ['a', 'b', 'c']; a.push('d'); // 4 (['a', 'b', 'c', 'd'])

Array.prototype.reduce()

arr.reduce(callback[, initialValue])

The reduce() method applies a function against an accumulator and each element in the array (from left to right) to reduce it to a single value.

['x', 'y', 'z'].reduce((v1, v2) => v1 + v2, 'abc'); // 'abcxyz'

Array.prototype.reduceRight()

arr.reduceRight(callback[, initialValue])

The reduceRight() method applies a function against an accumulator and each value of the array (from right-to-left) to reduce it to a single value.

['c', 'b', 'a'].reduceRight((v1, v2) => v1 + v2, 'xyz'); // 'xyzabc'

Array.prototype.reverse()

arr.reverse()

The reverse() method reverses an array in place. The first array element becomes the last, and the last array element becomes the first.

[1, 2, 3].reverse(); // [3, 2, 1]

Array.prototype.shift()

arr.shift()

The shift() method removes the first element from an array and returns that removed element. This method changes the length of the array.

a = [1, 2, 3]; a.shift(); // 1 ([2, 3])

Array.prototype.slice()

arr.slice([begin[, end]])

The slice() method returns a shallow copy of a portion of an array into a new array object selected from begin to end (end not included). The original array will not be modified.

['a', 'b', 'c', 'd'].slice(2, 3); // ['c']

Array.prototype.some()

arr.some(callback[, thisArg])

The some() method tests whether at least one element in the array passes the test implemented by the provided function.

[2, 5, 10, 12].some(v => v/2 === 4); // false

Array.prototype.sort()

arr.sort([compareFunction])

The sort() method sorts the elements of an array in place and returns the array. The sort is not necessarily stable. The default sort order is according to string Unicode code points.

[1, 30, 4, 21].sort(); // [1, 21, 30, 4]

Array.prototype.splice()

arr.splice(start[, deleteCount[, item1[, item2[, ...]]]])

The splice() method changes the contents of an array by removing existing elements and/or adding new elements. Returns deletion(s).

a = [1, 2, 3, 4]; a.splice(2, 2, 'a') // [3, 4] ([1, 2, 'a'])

Array.prototype.toString()

arr.toString()

The toString() method returns a string representing the specified array and its elements.

[1, 2, 'a', '1a'].toString(); '1,2,a,1a'

Array.prototype.unshift()

arr.unshift(element1[, ...[, elementN]])

The unshift() method adds one or more elements to the beginning of an array and returns the new length of the array.

a = [1, 2, 3]; a.unshift(4, 5); // 5 ([4, 5, 1, 2, 3])

Array.prototype.values()

arr.values()

The values() method returns a new Array Iterator object that contains the values for each index in the array.

for (let val of ['a', 'b', 'c'].values()) { console.log(val); }; // a, b, c

Object Methods

Object.assign()

Object.assign(target, ...sources)

The Object.assign() method is used to copy the values of all enumerable own properties from one or more source objects to a target object. It will return the target object.

Object.assign({}, {a: 0, b: 2, c: 2}, {b: 1, d: 3}); // { "a": 0, "b": 1, "c": 2, "d": 3}

Object.create()

Object.create(proto[, propertiesObject])

The Object.create() method creates a new object, using an existing object to provide the newly created object's __proto__ . Used to implement classical inheritance.

Object.create(Object.prototype); // {}

Object.defineProperties()

Object.defineProperties(obj, props)

The Object.defineProperties() method defines new or modifies existing properties directly on an object, returning the object.

Object.defineProperties({}, {a: { value: 0}}) // {a: 0}

Object.getOwnPropertyDescriptor()

Object.getOwnPropertyDescriptor(obj, prop)

The Object.getOwnPropertyDescriptor() method returns a property descriptor for an own property (that is, one directly present on an object and not in the object's prototype chain) of a given object.

Object.getOwnPropertyDescriptor({a: 0}, "a").value; // 0

Object.getOwnPropertyDescriptors()

Object.getOwnPropertyDescriptors(obj)

The Object.getOwnPropertyDescriptors() method returns all own property descriptors of a given object.

Object.getOwnPropertyDescriptors({a: 0}, "a"); // {a: {value: 0, writable: true, enumerable: true, configurable: true}}

Object.getOwnPropertyNames()

Object.getOwnPropertyNames(obj)

The Object.getOwnPropertyNames() method returns an array of all properties (including non-enumerable properties except for those which use Symbol) found directly upon a given object.

Object.getOwnPropertyNames({a: 0, b: 1, c: 2}); // ["a", "b", "c"]

Object.getOwnPropertySymbols()

Object.getOwnPropertySymbols(obj)

The Object.getOwnPropertySymbols() method returns an array of all symbol properties found directly upon a given object.

Object.getOwnPropertySymbols({}); // []

Object.getPrototypeOf()

Object.getPrototypeOf(obj)

The Object.getPrototypeOf() method returns the prototype (i.e. the value of the internal [[Prototype]] property) of the specified object.

proto = {}; obj = Object.create(proto); Object.getPrototypeOf(obj) === proto; // true

Object.is()

Object.is(value1, value2)

The Object.is() method determines whether two values are the same value.

Object.is("a", "a"); // true

Object.isExtensible()

The Object.isExtensible() method determines if an object is extensible (whether

it can have new properties added to it).

obj = Object.preventExtensions({}); obj.test = true; obj.test; // undefined

Object.isFrozen()

Object.isFrozen(obj)

The Object.isFrozen() determines if an object is frozen.

Object.isFrozen({a: 0}); // false

Object.isSealed()

Object.isSealed(obj)

The Object.isSealed() method determines if an object is sealed.

Object.isSealed({a: 0}); // false

Object.keys()

Object.keys(obj)

The Object.keys() method returns an array of a given object's own enumerable properties, in the same order as that provided by a for...in loop (the difference being that a for-in loop enumerates properties in the prototype chain as well).

Object.keys({a: 0, b: 1, c: 2}); // ["a", "b", "c"]

Object.preventExtensions()

Object.preventExtensions(obj)

The Object.preventExtensions() method prevents new properties from ever being added to an object (i.e. prevents future extensions to the object).

obj = Object.preventExtensions({}); Object.defineProperty(obj, "key", {value: true}); // TypeError

Object.prototype.hasOwnProperty()

obj.hasOwnProperty(prop)

The hasOwnProperty() method returns a boolean indicating whether the object has the specified property as own (not inherited) property.

var obj = {a: 0}; obj.hasOwnProperty("a"); obj.hasOwnProperty("toString"); // true, false

Object.prototype.isPrototypeOf()

prototypeObj.isPrototypeOf(object)

The isPrototypeOf() method checks if an object exists in another object's prototype chain.

Object.prototype.isPrototypeOf(Array) // true

Object.prototype.propertyIsEnumerable()

obj.propertyIsEnumerable(prop)

The propertyIsEnumerable() method returns a Boolean indicating whether the specified property is enumerable.

([1,2,3]).propertyIsEnumerable("length")

Object.prototype.toLocaleString()

obj.toLocaleString()

The toLocaleString() method returns a string representing the object. This method is meant to be overridden by derived objects for locale-specific purposes.

(1234.5678).toLocaleString("de-DE"); // "1.234,568"

Object.prototype.toString()

obj.toString()

The toString() method returns a string representing the object.

[1,2,3].toString(); // "1,2,3"

Object.prototype.valueOf()

object.valueOf()

The valueOf() method returns the primitive value of the specified object.

(new Date()).valueOf(); // date coerced into timestamp

Object.seal()

Object.seal(obj)

The Object.seal() method seals an object, preventing new properties from being added to it and marking all existing properties as non-configurable. Values of present properties can still be changed as long as they are writable.

obj = Object.seal({a: 1}); obj.b = 2; obj; // {a: 1}

Object.setPrototypeOf()

Object.setPrototypeOf(obj, prototype);

The Object.setPrototypeOf() method sets the prototype (i.e., the internal [[Prototype]] property) of a specified object to another object or null.

dict = Object.setPrototypeOf({}, null); dict.a = 0; // {a: 1}

Object.values()

Object.values(obj)

The Object.values() method returns an array of a given object's own enumerable property values, in the same order as that provided by a for...in loop (the difference being that a for-in loop enumerates properties in the prototype chain as well).

Object.values({a: 0, b: 1, c: 2}); // [0, 1, 2]
